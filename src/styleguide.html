3.3 Responsive Breakpoint System

Standardized breakpoint system for consistent responsive design. All breakpoints use rem units for better accessibility and are defined as CSS Custom Properties in src/css/variables.css.

Breakpoint Reference

--breakpoint-xs: 20rem (320px) - Small phones
--breakpoint-sm: 36rem (576px) - Large phones
--breakpoint-md: 48rem (768px) - Tablets
--breakpoint-lg: 64rem (1024px) - Desktop
--breakpoint-xl: 80rem (1280px) - Large desktop
--breakpoint-2xl: 96rem (1536px) - Extra large

3.4 Spacing System

Consistent spacing is key to a well-designed interface. We use a spacing scale based on multiples of a base unit (e.g., 4px or 8px). This is implemented using CSS Custom Properties in src/css/variables.css.

Examples:

--spacing-xs (e.g., 4px)
--spacing-sm (e.g., 8px)
--spacing-md (e.g., 16px)
--spacing-lg (e.g., 24px)
--spacing-xl (e.g., 32px)

These variables should be used for margins, paddings, and gaps to maintain vertical and horizontal rhythm.

3.5 Unified Grid System

A comprehensive, mobile-first, responsive grid system designed for creating flexible and consistent layouts. The system is built with CSS Grid and includes utility classes for columns, gaps, and responsive visibility.

Core Usage

To create a grid, add the .grid class to a container. Then, add modifier classes to define the column structure and gap size. 

<div class="grid grid--3-column grid--gap-m">
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3</div>
</div>

Gap Variations

Control the space between grid items using gap utilities. The default gap is --space-l.

  • .grid--gap-xs
  • .grid--gap-s
  • .grid--gap-m
  • .grid--gap-l
  • .grid--gap-xl
  • .grid--gap-2xl
  • .grid--gap-3xl

Responsive Visibility Utilities

Show or hide elements at specific breakpoints.

  • .hide-mobile: Hidden below 48rem.
  • .hide-tablet: Hidden between 48rem and 64rem.
  • .hide-desktop: Hidden above 64rem.
  • .hide-until-lg: Hidden until 64rem.
  • .hide-until-xl: Hidden until 80rem.
  • .hide-until-xxl: Hidden until 96rem.
